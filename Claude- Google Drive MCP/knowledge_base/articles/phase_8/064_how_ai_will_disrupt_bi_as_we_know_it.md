## Summary
Tristan Handy argues that AI will fundamentally restructure BI by disaggregating its three core functions—semantic modeling, exploratory data analysis, and presentation—allowing AI agents to own the EDA phase while BI tools retain governance and production artifact management. This matters for data leaders because it signals a coming shift in where analytical work originates and how governance pipelines must be redesigned to accommodate AI-driven exploration feeding into governed, production-grade reporting.

## Key Excerpts
> BI tools serve three distinct jobs: semantic modeling, exploratory data analysis, and presentation — and AI is poised to disrupt each differently.
> The "production line" of BI moves artifacts from raw exploration through personal reporting, shared reporting, and finally governed production assets.
> For the BI production pipeline to work, EDA must begin inside the BI tool — but BI tools are rarely the best environment for that work.
